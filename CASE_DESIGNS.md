# 3D Case Designs

The 3D printable case designs for Stratux have been moved to a dedicated repository to keep the main repository lean.

**📁 [View all case designs →](https://github.com/stratux/stratux-cases)**

The separate repository includes:
- Long version cases with fan options
- Nano/Nanopi compact cases
- Pi Zero cases
- Single-board configurations
- Three-layer stacked cases
- Various mounting solutions (antenna, RAM, suction cup, GoPro-style)

All designs are available in multiple formats:
- `.STL` – Print-ready files
- `.STEP` – CAD exchange format
- `.SLDPRT` / `.SLDASM` – SolidWorks source files
- Reference images

This change reduces the main Stratux repository size and makes case designs easier to maintain and discover.
